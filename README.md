# Insta Love Button

A simple web application that demonstrates a heart animation effect when double-clicking on an image. This project uses HTML, CSS, and JavaScript to create a visually appealing "like" button effect.

## Features

- Displays an image with a heart icon overlay.
- Animates the heart icon to "pop" when double-clicked.
- Utilizes CSS transitions and JavaScript for interaction.

## Files

- `index.html`: The main HTML file containing the structure and script for the animation.
- `love.css`: (Optional) Additional CSS file for custom styling if needed.
- `README.md`: This file, containing project information and instructions.
